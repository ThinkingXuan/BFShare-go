# BFShare-go
Secure File Outsourcing and Sharing Method Based on Alliance Blockchain 基于联盟链的文件安全外包与共享方法

功能：


- 初始化
```shell
./BFShare init
```

- 上传
```shell
./BFShare upload -p 文件路径
```

- 展示
```shell
./BFShare show 
```

- 分享
```shell
./BFShare share -i  文件ID  -a 文件接收方地址
```

- 下载
```shell
./BFShare download -i 文件ID  -d 文件存储路径
```
